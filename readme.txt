Custom skeduled Posts. This widget gives you the ability to show  skeduled posts for start and end times on the widget content.

== Description ==

This plugin gives users the posibility to set a start and end date for each post in a custom form at the bottom of each post and then to add a widget to a sidebar that will show posts in the specified intervals accourding to the current date


== Installation ==

You either do it:

1. The hard way:

 upload it to the plugins directory

2. the easy way

Upload the zip to the plugins admin panel wordpress page on your wordpress site


== Frequently Asked Questions ==

= Is there a post limit? =

No

= How can I delete the date intervals? =

You just set them to a different start or end date smaller than the current date. They will never show.
